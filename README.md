## 프로젝트 소개 

회원 및 관리자가 만든 챌린지에 참여하기 위해 미리 결제한 예치금을 사용하거나 챌린지 참여 시 바로 결제하여 챌린지에 참여할 수 있다.
달성률에 따라 예치금을 환급하여 100% 달성 시 상금을 추가적으로 줌으로써 좋은 습관을 만들 수 있도록 돕는 서비스다.
